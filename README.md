## 🚀 How to Put This Online (3 steps)

### Step 1 — Upload to GitHub
1. Go to [github.com](https://github.com) → sign in → click **"New repository"**
2. Name it anything (e.g. `nova-strike`), set to **Public**, click **Create repository**
3. On the empty repo page click **"uploading an existing file"**
4. **Drag your entire game folder** into the upload box and wait for it to finish
5. Click **"Commit changes"**

### Step 2 — Enable GitHub Pages
1. In your repo go to **Settings** (top menu)
2. Click **Pages** (left sidebar)
3. Under **Source** select **"GitHub Actions"**
4. Click **Save**

### Step 3 — Wait ~1 minute
The game deploys automatically. Your link will be:
`https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/`

> The GitHub Action in `.github/workflows/deploy.yml` handles everything automatically — it finds your game files no matter what folder they're in and serves them correctly.
